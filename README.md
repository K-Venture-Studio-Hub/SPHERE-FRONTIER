## What Is K Venture Studio Hub?

K Venture Studio Hub bridges learning into venture building. It provides pre-prepared business models, canvas templates, and startup acceleration scaffolds derived from Science, Entrepreneurship, and Technology directions.

Inspired by frameworks from Kyiv School of Economics and MIT Orbit, adapted for the SET ecosystem.

## Core Repositories

| Repository | Focus |

| [`SPHERE-I-SCIENCE`](https://github.com/K-Venture-Studio-Hub/SPHERE-I-SCIENCE) | Venture models for science-based startups (biotech, green, agtech, edtech) |
| [`SPHERE-II-ENTREPRENEURSHIP`](https://github.com/K-Venture-Studio-Hub/SPHERE-II-ENTREPRENEURSHIP) | Canvas templates, validation frameworks, go-to-market architectures |

- 🏗️ **Business Model Canvas** — pre-filled templates per SET direction
